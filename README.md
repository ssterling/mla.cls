mla.cls
=======

LaTeX class for MLA papers.

I'll write a README later; for now, refer to the
hodge-podge documentation:
```console
$ latex mla.dtx
$ gv mla.dtx
```

Installation
------------

Standard:
```console
$ latex mla.ins
$ mv mla.cls ~/texmf/tex/latex/mla.cls
```
